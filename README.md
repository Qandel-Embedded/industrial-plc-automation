# Industrial PLC Automation System

Industrial automation project featuring PLC programming (IEC 61131-3), SCADA integration, Modbus RTU communication, and custom HMI design.

## System Components
- **PLC:** Siemens S7-1200 / Allen Bradley CompactLogix
- **SCADA:** Wonderware / custom Python SCADA
- **Communication:** Modbus RTU, Profibus, EtherNet/IP
- **HMI:** Custom operator panels + WinCC

## Features
- Real-time process monitoring
- Alarm management system
- Historical data logging
- Remote access via VPN
- IEC 61131-3 compliant code (Ladder, FBD, ST)

- 15% increase in production efficiency
- 30% reduction in unplanned downtime
